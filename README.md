# Graphene Flask Tutorial

```shell
mkdir graphene-flask-tutorial
cd graphene-flask-tutorial
hub init -g
hub create
poetry init
poetry lock
poetry add sqlalchemy
poetry add graphene-sqlalchemy
poetry add flask
poetry add flask-graphql
```
